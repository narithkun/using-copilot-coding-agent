# Mergington High School Activities

A website application that allows students to view extracurricular activities and teachers to manage student registrations.

## Features

- View all available extracurricular activities
- Search activities by name
- Filter activities by category (Sports, Arts, Academic, Community, Technology)
- Filter activities by day (Monday through Sunday)
- Filter activities by time (Before School, After School, Weekend)
- Teacher login system for managing student registrations
- Sign up students for activities (requires teacher authentication)
- Unregister students from activities (requires teacher authentication)
